Introduction

What is a Function​

A function is relationship that maps one input to one output. Inputs are almost exclusively xxx as input and yyy as output. Written as a function:

y=f(x)\begin{equation} y=f(x) \end{equation}y=f(x)​​

The input or xxx is often refered to as an independent variable. The output, yyy or f(x)f(x)f(x) is often refered to as a dependent variable.

What Makes a Function​

When you input a number, the output cannot be more than one. For example:

f(1)=2f(1)=3\begin{equation} \begin{aligned} f(1) = 2 \\ f(1) = 3 \end{aligned} \end{equation}f(1)=2f(1)=3​​​

The example above means this is not a function.
